# AHC: Meta-Learned Adaptive Compression for Continual Object Detection on Memory-Constrained Microcontrollers

**AHC (Adaptive Hierarchical Compression)** is a novel [[Meta-Learning]] framework designed to enable effective [[Continual Learning]] for [[Object Detection]] on extreme-edge hardware, specifically [[Microcontrollers]] (MCUs) with memory capacities limited to under 100KB.
