# XR-CareerAssist

XR-CareerAssist is an innovative platform designed to transform traditional, static career guidance into an immersive, interactive experience. Unlike conventional text-driven systems that struggle to engage users, XR-CareerAssist utilizes [[interact-an-ai-driven-extended-reality-framework-for-accesible-communication-fea|Extended Reality]] (XR) and [[steering-the-verifiability-of-multimodal-ai-hallucinations|Multimodal AI]] to provide personalized, evidence-based career insights.

### Technical Architecture
The platform's architecture integrates five distinct [[artificial-intelligence-and-the-structure-of-mathematics|Artificial Intelligence]] modules within a single 3D environment to create a seamless, multi-sensory user experience. These components include:

*   **Automatic Speech Recognition (ASR):** Enables intuitive, voice-driven interaction.
*   **Neural Machine Translation:** Provides multilingual support across English, Greek, French, and Italian.
*   **Langchain-based Training Assistant:** A conversational agent that delivers personalized dialogue using advanced [[from-large-language-model-predicates-to-logic-tensor-networks-neurosymbolic-offe|Large Language Model]] frameworks.
*   **BLIP-based Vision-Language Model:** Utilized for generating complex, automated career visualisations.
*   **AWS Polly Text-to-Speech:** Delivers high-quality audio through an interactive 3D avatar.
