# General Explicit Network (GEN)

The **General Explicit Network (GEN)** is an innovative [[benchmarking-deep-learning-for-future-liver-remnant-segmentation-in-colorectal-l|Deep Learning]] architecture specifically engineered to advance the field of solving [[ae-vit-stable-long-horizon-parametric-partial-differential-equations-modeling|Partial Differential Equations]] (PDEs). As computational science moves toward more integrated models, GEN provides a new framework to address the shortcomings of existing neural solvers.
