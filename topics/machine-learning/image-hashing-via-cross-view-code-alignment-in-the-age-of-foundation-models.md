# Image Hashing via Cross-View Code Alignment in the Age of Foundation Models

The paper "Image Hashing via Cross-View Code Alignment in the Age of Foundation Models" introduces a novel framework, **CroVCA** (Cross-View Code Alignment), designed to address the computational bottlenecks inherent in large-scale image retrieval.
