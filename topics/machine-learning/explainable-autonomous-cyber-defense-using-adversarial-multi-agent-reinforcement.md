# Explainable Autonomous Cyber Defense using Adversarial Multi-Agent Reinforcement Learning

The research introduces the **Causal Multi-Agent Decision Framework (C-MADF)**, a novel architecture designed to strengthen [[claw-eval-toward-trustworthy-evaluation-of-autonomous-agents|Autonomous Agents]] used in critical infrastructure defense. As [[advanced-persistent-threats|Advanced Persistent Threats]] (APTs) increasingly employ "Living off the Land" techniques and telemetry-based perturbations, traditional [[a-comparative-analysis-of-machine-learning-models-in-shap-analysis|Machine Learning]] defense systems are prone to reasoning drift, overreaction, and the misclassification of benign activity.
