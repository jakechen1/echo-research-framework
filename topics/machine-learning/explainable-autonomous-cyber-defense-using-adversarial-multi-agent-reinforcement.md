# Explainable Autonomous Cyber Defense using Adversarial Multi-Agent Reinforcement Learning

The research introduces the **Causal Multi-Agent Decision Framework (C-MADF)**, a novel architecture designed to strengthen [[Autonomous Agents]] used in critical infrastructure defense. As [[Advanced Persistent Threats]] (APTs) increasingly employ "Living off the Land" techniques and telemetry-based perturbations, traditional [[Machine Learning]] defense systems are prone to reasoning drift, overreaction, and the misclassification of benign activity.
