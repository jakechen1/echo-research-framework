# ResearchEVO

**ResearchEVO** is a novel end-to-end framework designed to automate the entire lifecycle of scientific progress, from initial experimentation to formal documentation. The framework is built upon the "discover-then-explain" paradigm, which seeks to computationally replicate the human scientific process: an initial period of undirected experimentation to find unexpected phenomena, followed by a retrospective analysis to integrate those findings into existing theoretical frameworks.

## Framework Architecture

The ResearchEVO pipeline consists of two primary operational stages:

### 1. The Evolution Phase
This phase focuses on algorithmic discovery through [[large-language-models-for-outpatient-referral-problem-definition-benchmarking-an|Large Language Models]] (LLMs). It employs a process known as bi-dimensional co-evolution, where the system simultaneously optimizes both the underlying algorithmic logic and the overall structural architecture. This search is driven purely by "fitness," allowing the system to navigate complex code-implementation spaces and identify high-performing solutions without requiring a prior understanding of the solutions' mechanics.

### 2. The Writing Phase
Once the most effective algorithm is identified, the framework transitions into a retrospective documentation mode. Utilizing sentence-level [[contradictions-in-context-challenges-for-retrieval-augmented-generation-in-healt|Retrieval-Augmented Generation]] (RAG), the system generates a complete, publication-ready [[latex|LaTeX]] manuscript. To maintain scientific rigor, the framework incorporates:
* **Anti-hallucination verification:** Explicit checks to ensure no false information is presented.
* **Automated experiment design:** The ability to structure new experiments to validate findings.
* **Literature grounding:** Ensuring all "blind" discoveries are appropriately situated within the context of existing scientific literature.

## Empirical Validation

The researchers validated ResearchEVO across two distinct scientific domains:
* **[[quantum-error-correction|Quantum Error Correction]]**: Utilizing real-world quantum hardware data from Google.
* **[[curvature-aware-optimization-for-high-accuracy-physics-informed-neural-networks|Physics-Informed Neural Networks]] (PINNs)**: Testing the application of [[a-comparative-analysis-of-machine-learning-models-in-shap-analysis|Machine Learning]] to physical laws.

In both instances, the Evolution Phase successfully discovered new, human-interpretable algorithmic mechanisms that had not been previously proposed in scientific literature. Furthermore, the Writing Phase successfully bridged these discoveries with established theory, producing accurate, compilable documents with zero fabricated citations.
