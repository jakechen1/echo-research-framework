# Probing 3D Chromatin Structure Awareness in Evo2 DNA Language Model

The study "Probing 3D Chromatin Structure Awareness in Evo2 DNA Language Model" investigates the ability of large-scale [[DNA language models]] to capture the complexities of [[chromatin structure]]. While the [[Evo2]] model features a massive million-token context window—theoretically wide enough to encompass entire [[Topologically Associating Domains (TADs)]]—the research questions whether the model has truly internalized the 3D regulatory layers that function atop the primary DNA sequence.
