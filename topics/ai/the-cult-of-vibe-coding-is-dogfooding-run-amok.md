# The cult of vibe coding is dogfooding run amok

"Vibe coding" is a polarizing term emerging within the [[Software Engineering]] community. It describes a methodology where developers rely primarily on the "vibe"—the high-level intent or intuitive description—provided to [[Large Language Models]] (LLMs) to generate executable code. This approach suggests a paradigm shift from traditional, logic-driven syntax construction toward a descriptive, natural-language-driven workflow.
