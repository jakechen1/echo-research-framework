# LAsset: An LLM-assisted Security Asset Identification Framework for System-on-Chip (SoC) Verification

LAsset is an innovative automated framework designed to streamline the identification of security assets within complex hardware architectures. As modern [[System-on-Chip (SoC)]] and [[Intellectual Property (IP)]] designs grow in complexity, ensuring robust security becomes increasingly difficult.
