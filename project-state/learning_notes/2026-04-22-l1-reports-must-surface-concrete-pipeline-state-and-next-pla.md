# Learning: L1 reports must surface concrete pipeline state and next planned step, not just scavenger row count delta

## Context
User feedback Apr 22 2026 2:23 PM: 'Reports cannot be no change.' L1 report template currently keys on scavenger row count only; must also include current task+stage, last artifact produced, and next executor scheduled.

## Enforcement
This rule should be checked during Assessor (A-stage) reviews and
enforced by relevant skill upgrades. Future iterations must not repeat
the pattern that triggered this correction.
