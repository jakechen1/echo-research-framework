# 🧬 Project: PHGDH-Allosteric-RBD-Binder

## 🎯 Project Objective
To discover and validate novel allosteric modulators for **Phosphoglycerate Dehydrogenase (PHGDH)** using a bimodal discovery architecture: **Docking-Centric (Physics-based)** and **LM-Centric (Generative Intelligence)**.

## 🏗️ Research Architecture: Bimodal Strategy

This project employs two parallel computational tracks to ensure both structural accuracy and generative innovation.

### ⚓ Track 1: The Anchor (Docking-Centric)
**Role:** High-fidelity structural validation using physics-based simulation.
*   **Methodology:** High-throughput molecular docking (Smina/AutoDock Vina) and molecular dynamics (MD) refinement.
*   **Function:** Serves as the "Ground Truth" to validate hits generated by the generative track.

### 🚀 Track 2: The Engine (LM-Centric)
**Role:** Target-agnostic generative evolution using chemical language models.
*   **Methodology:** Sequential fine-tuning of chemical language models (e.g., ChemBERTa/LSTM) on potency-sorted SMILES datasets.
*   **Function:** Rapidly expands the chemical space by generating high-affinity candidates via "Perplexity-as-Potency" prediction.
