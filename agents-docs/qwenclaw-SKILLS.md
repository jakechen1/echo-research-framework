reading/writing project-state files. But when needed, these are available:

- `cheaha-hpc` — if a sub-goal involves Cheaha job submission, read this
  first and draft the sbatch for jakeclaw to submit.
- `terminal-command-execution` — for shaping commands sent to jakeclaw.
- `wiki-builder` (Hermes) — when a sub-goal involves wiki growth, invoke
  Hermes' wiki_builder.py directly; don't ask jakeclaw.
